注意力小实验

想了解自己注意力到底有多厉害吗？想改善自己的注意力吗？

就来进行一个注意力小测试吧，实验总共分成三个阶段，

按顺序一个一个完成后可得实时奖励金额十块钱

实验介绍及实验流程

本实验将探索您的注意力，根据下方实验流程逐步进行实验，测量您的注意程度，给出关于您注意力的研究报告，以及您完成实验后还可以额外获得10块钱的红包作为奖励。

Icon

1. 简易的个人信息调查

将会收集您些许必要的个人资料，如年龄、生活地区等，有助进行研究的分类。

Icon

2. 个人特质调查

如实作答问题使得研究者可以对您的个人特质进行深入的了解，并结合注意力实验进行研究。

Icon

3. 影片观看

随机抽取一段影片，请您细心了解影片内容。

Icon

4. 注意力实验

进行简单的注意力实验，请根据实验指导语完成整个实验。

Icon

5. 回忆测验

您需要回答部分与影片相关的内容，以测试您观看影片时候的注意程度。

Icon

领取您的奖励

完成所有实验后，领取属于您的注意力报告及10块钱奖励。
